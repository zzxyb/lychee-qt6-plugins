# lychee-qt6-plugins
lychee desktop environment qt6 plugin collection

### Compiling from Source
Install dependencies:

build dependencies(for arch linux):
* cmake
* extra-cmake-modules
* qt6-base
* qt6-wayland
* qt6-svg

build dependencies(for debian):
* g++
* cmake
* extra-cmake-modules
* qt6-base-dev
* qt6-base-private-dev
* qt6-wayland-dev
* qt6-wayland-private-dev
* qt6-svg-dev
